# A2A :sunglasses:

**A2A Protocol Version : 0.3.26**:white_check_mark:<br><br> 
**Elon's Agent is the client agent or host agent, which is sending the request to Mark and Jeff's Agent**:snowflake:<br>
**Mark's Agent is developed using CrewAI**:green_heart:<br>
**Elon's agent is developed using Google ADK**:purple_heart:<br>
**Jeff's Agent is developed using Langchain**:blue_heart:<br>
