# A2A
**Personal Agent of Elon Musk, Jeff Bezos and Mark Zuckerberg communicating using A2A to find available time slot to schedule tennis match**<br><br>
**A2A Protocol Version : 0.3.26**<u><br><br>
**Elon's Agent is the client agent or host agent, which is sending the request to Mark and Jeff's Agent**<br>
**Mark's Agent is developed using CrewAI**<br>
**Elon's agent is developed using Google ADK**<br>
**Jeff's Agent is developed using Langchain**<br>
